Controladores Industriales PLC

Se utilizó el software Studio 5000 y el lenguaje de programación Ladder para elaborar el programa de control de las máquinas automatizadas.


Máquina para corte de bambú.

El programa correspondiente a la máquina de corte de bambú funciona de la siguiente forma:

  • 0. Mediante una señal de inicio se activa y se enclava la banda transportadora.
  • 1. Al activarse el sensor de la banda transportadora se activa un temporalizador, el cual cuenta el tiempo requerido para que el tubo se ubique a una determinada distancia siendo trasladado a la velocidad de la banda transportadora.
  • 2. La banda también se puede activar cuando se completa una vez la rutina diseñada.
  • 3. Una vez el temporizador finaliza y la banda se detiene, se activa el cilindro del sistema de sujeción. Se da una espera mientras es ajustado el tubo.
  • 4. Cumplido el tiempo para ajustar, se activa el cilindro y el motor de la sierra para proceder con el corte del material.
  • 5. Cuando el cilindro se extiende completamente el proceso de corte finaliza y el cilindro inmediatamente se contrae, retornando la sierra a su posición inicial. Se utiliza un contador para diferenciar la posición inicial antes y después de realizar el corte.
  • 6. Luego el cilindro sistema de sujeción se expande para soltar el tubo y apenas llega a su extensión total inicia un nuevo proceso de corte.

Máquina para corte tipo boca de pescado y tipo transversal en tubos de aluminio.

La parte inicial de programa de control perteneciente a la máquina de boca de pescado es muy similar al de la máquina de corte de bambú. En la línea 3 se diferencia en que mientras se hace la sujeción del tubo, el posicionador va ubicando la herramienta copa sierra en ángulo de corte deseado. En este caso primero se activa el copa sierra, corta, regresa y luego se activa la sierra. En la línea 8 cuando ambos procesos han finalizado el cilindro del sistema de sujeción se expande, liberando así el tubo para proceder con el siguiente corte.
